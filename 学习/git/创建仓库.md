###### 忽略文档
在文档那里右击鼠标，点击Ignore file 
![[Pasted image 20250113171842.png]]
如果是想要忽略掉所有以.bak结尾的文档，就在文档那里右击鼠标，点击Ignore all .bak
那么再下次有新建的以.bak结尾的文档，软件将不会对它进行比对
