1.下载miniconda安装脚本
```
wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh
```
2.运行安装脚本
```
bash Miniconda3-latest-Linux-x86_64.sh
```
3.按enter键，输入yes同意协议。按enter，使用默认路径或自定义。输入yes，就会自动把conda添加到PATH
4.激活环境
关闭重新打开终端或运行
```
source ~/.bashrc
```
5.验证安装
```
conda --version
python --version
```
6.使用
```
python xxx.py
```
